# 🌍 TariffIQ – Walmart Sparkathon Project

**TariffIQ** is a supply chain analytics web application built for the Walmart Sparkathon challenge. It helps procurement teams understand how import tariffs impact supplier costs and supports cost optimization decisions.

---

## 🔧 Features

- 📦 View and manage suppliers
- 🌐 Filter by product and country
- 💰 Calculate cost impact based on new tariffs
- 🧮 Identify top suppliers by cost
- ➕ Add new suppliers to the system
- 📊 Optimize supplier choice for better pricing

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask), Pandas
- **Frontend:** HTML, CSS (Bootstrap), JavaScript
- **Storage:** CSV (for supplier data)
- **Hosting:** Render / GitHub Pages (Frontend)
